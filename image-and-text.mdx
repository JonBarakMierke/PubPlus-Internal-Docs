title: "Image & Text"

A two-column section pairing an image with a heading, body copy, and an optional CTA. Renders as `<x-sections.image-and-text>`.

## File Structure

```text
resources/views/components/sections/image-and-text/
└── index.blade.php   # Full component (no variants)
```

---

## Props

| Prop | Type / Values | Default | Notes |
| :-- | :-- | :-- | :-- |
| `image` | image URL | `null` | Source for the image column. **Always pass one** — an empty value renders a broken image. |
| `imageSide` | `left` \| `right` | `left` | Which side the image sits on (`right` flips to `md:flex-row-reverse`). |
| `aspect` | see [Aspect Ratios](#aspect-ratios) | `video` | Aspect ratio of the image frame. |
| `container` | any Tailwind `max-w-*` class | `max-w-7xl` | Max width of the section content. |
| `ctaType` | `button` \| `link` | `button` | How the `cta` slot renders. |
| `ctaLink` | URL | `/` | Destination for the CTA. |

Kebab-case in Blade tags, e.g. `image-side="right"`, `cta-type="link"`, `cta-link="/about-us"`.

---

## Slots

| Slot | Required | Description |
| :-- | :-- | :-- |
| `heading` | No | Heading, rendered via `x-ui.heading` at `size="xl"`. |
| _(default)_ | No | Body content — wrapped in `x-ui.text`. Renders as a `<div>`, so it's safe to nest `<p>` and `<ul>` inside. |
| `cta` | No | CTA label. Renders as an `x-ui.button` (or `x-ui.link` when `cta-type="link"`) pointing at `ctaLink`. |

---

## Aspect Ratios

| Value | Result |
| :-- | :-- |
| `square` | `aspect-square` |
| `portrait` | `aspect-[3/4]` |
| `landscape` | `aspect-[4/3]` |
| `video` | `aspect-video` (default) |
| `wide` | `aspect-[21/9]` |
| `none` | No aspect wrapper — image renders at its natural height. |
| _(custom)_ | Any raw Tailwind aspect class, e.g. `aspect="aspect-[2/1]"`. |

## Usage Examples

### 1. Image left, heading \+ body \+ button

        <span class="font-lemonmilk font-bold text-neutral-900 dark:text-neutral-50">Built for your business</span>

## Notes

- **Alternate sides** down a page (`left`, then `right`, then `left`, …) for a natural zig-zag rhythm.
- The default slot is wrapped in `x-ui.text` (a `<div>`), so paragraphs and lists nest cleanly — no invalid `<p><ul></p>` nesting.
- There are no variants or themes; the section is transparent and inherits the surrounding page surface.
